# 它本来的用途

亚硝酸钠作为食品添加剂，不仅可以对鱼类、肉类食品进行染色和保鲜，原理是：亚硝酸盐在酸性条件下分解产生亚硝基，亚硝基会很快与肌红蛋白反应生成稳定、鲜艳、亮红色的亚硝化肌红蛋白，使肉制品保持稳定的鲜艳红色。因此特别适用于午餐肉等。
亚硝酸钠还可以抑制肉毒杆菌的生长。后者可以产生肉毒杆菌毒素，造成食物中毒。在欧盟标准下，亚硝酸钠只能以不高于0.6%的含量与食用盐混合使用。

# 但却被用作了毒药

其价格在15元/100g左右（tb），100g已经远远超过致死量。
亚硝酸钠对鼠的半数致死量为180 mg/kg，对人的已知最低致死量为71 mg/kg，就是说一个重60kg的人可能被最低4.2g该种物质致死。
亚硝酸盐致死的机制是，通过结合血红蛋白窒息。
同时，亚硝酸钠与胃酸反应会生成会生成亚硝酸和氯化钠，亚硝酸不稳定，会进一步分解为一氧化氮，二氧化氮等有毒气体。

# 无痛死亡？

中毒时，皮肤和各器官出现蜡黄至青紫等表征缺氧的颜色。
服用后，大约会在一段时间后（15min-2h不等）失去意识，在此之前，会有一定程度的室息感。
由于属于缺氧直接导致的死亡，没有对神经系统压制，濒死时脑干会强烈唤醒，会有几分钟的全身剧痛。唤醒的效果也包括剧烈心跳和血管爆裂。（存在争议）
大部分自杀者会选择凌晨服用，4-6小时内无人干预基本可以确定死亡。

# 报警

首先，将亚硝酸钠等毒药储存在身边（尤其是卧室内）会大大增加冲动自杀的概率。如果您发现有朋友这么做，请尽可能劝说他将药倒掉。
如果对方确凿服用了自杀药物，且处于失联状态，应当选择报警。 

**收集个人信息**
（1）查询社交媒体。尽可能查阅对方在社交媒体（QQ空间、朋友圈、推）来获取上述信息。
（2）询问朋友。尽可能添加对方的现实朋友或网络朋友，以询问方式获取个人信息。
（3）查询各平台信息。尽可能的通过对方发布的文字信息，照片，视频，快递订单，网购记录，历史聊天记录等各个方面的信息间接检索其个人信息。
（4）如果只有一个账号而没有其他个人信息的话。求助“开盒”的黑色产业链也未尝不可。

**电话报警**
（1）查询对方所在地的区号。可通过百度等搜索引擎搜索城市区号，如北京区号为010，之后拨打“010 110”。不加区号只会打到你自己所在地的公安！
（2）向警方提供对方的信息。包括但不限于身份证号，身份证姓名，身份证性别，电话号码，住址，社交关系，亲属联系方式，微信号，QQ号，工作单位，学校等，包括网络订单，快递订单，有多少信息给多少信息。更多的信息有助于增加警方确定对方身份的可能性，但不要提供无法确定真实性的信息。
（3）以大众能接受的方式提示跨性别或od信息。切记你的目的是救人，而非对跨性别概念进行科普，不要因和民警纠结跨性别概念而耽误宝贵的救助时间！
（4）特殊情况：只有QQ和微信号。若民警对此表示疑问，可用话术：我已经和我自己这里的公安报过警，对方说可以用数据手段，只需要提供QQ/微信号就可以了。如果民警拒绝受理，则可以通过微信进行再次报警。

**微信报警**

（当只有对方微信和QQ号没有其他任何信息的时候，微信报警可以以更少的信息成功报警，但是时间会相对较慢）
（1）搜索对方所在地的公安微信公众号。
（2）填写相关信息。最好把宣告自杀的帖子截图下来，在提供图片那一栏里发上去；也会要求提供你的电话号码（往往是需要短信验证码验证后才能报案）。
（3）民警来电核实。报警不久，民警会来电核实所给信息，也需要提供你的个人信息，如身份证号。

**也可以通过微信小程序 腾讯微光行动 报警。**
流程：微信小程序→腾讯微光行动→TA需要帮助→紧急情况上报。

# 急救
迅速就医，急诊注射美兰（亚甲基蓝注射液）。

治疗作用的大剂量、高浓度亚甲蓝具有氧化作用，直接生成高铁血红蛋白（MetHb），可能会快速加重病情。对于亚硝酸盐中毒，低剂量1-2mg/kg，必须缓慢静推（5-10分钟），并注意给药间隔时间。

使用了亚甲蓝之后，全身体液会呈现淡蓝色。因此把胃液澄清作为洗胃终点可能不是合理的。

VC具有强烈的还原作用，有促使高铁血红蛋白转变为血红蛋白的作用，在救治亚硝酸盐中毒时可能具有增效作用，但临床使用后确实有效。

注：报警或急救都是治标不治本的，为避免对方再次自杀，应当进行更长期的沟通与行动，但这不在本文的讨论范畴内。
