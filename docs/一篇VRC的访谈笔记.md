## VRC与现实的相同之处

- 都是在扮演角色。
- VRC的很多内容来源现实，高于现实。
- 个体来往于VRC和现实之间，存在和意识一脉相承。
## VRC与现实的相异之处

| VRC             | 现实                  |
| --------------- | ------------------- |
| 无时空限制           | 有时空限制               |
| 换人来脱离厌烦         | 换空间来脱离厌烦            |
| 变化很快            | 变化（像死了一样的）慢         |
| 自由度高            | 自由度低，社会限制（规范）很多     |
| 扮演角色压力小         | 扮演角色压力大             |
| 更换交往对象方便，脱钩容易   | 更改交往对象不方便，前任会产生持续影响 |
| 目的模糊地交流         | 目的明确地交流             |
| 试错成本低，影响短暂：“短痛” | 试错成本高，影响持久：“长痛”     |
## VRC与现实的双向影响

- 同时存在，同等重要→无法长时间脱钩
- 感觉A比B更美好→存在感与归属感的转移

具体而言：
i. 失去期待，不再奢望；
ii. 一切从简，降低物欲；
iii. 孤独和朋友的流失。

## 简单的感想

为什么会出现这些情况？存在决定意识，社会的形态决定人们的感受。
